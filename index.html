  • Test all links after pushing to GitHub

üí° Implementation Guide

To create subpages for your courses:

  1. In your GitHub repository, create a folder for each course (e.g., "course1", "python-ai")
  2. Add an index.html file to each folder with the course content
  3. Update the links on your main page to point to these folders
  4. Use relative paths for all assets (images, CSS, JavaScript)
  5. Test the links after pushing to GitHub
